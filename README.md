# 🏠 House Price Prediction

## 📌 Project Overview
This project predicts house prices using machine learning models. Multiple models were implemented and compared to select the best-performing one.

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  

---

## 📊 Models Used
- Linear Regression  
- KNN  
- Decision Tree  
- Random Forest  
- XGBoost ⭐ (Best Model)

---

## 📈 Results
- R² Score: **0.85**  
- MAE: **0.15**  
- RMSE: **0.21**  

---

## 🔍 Key Insights
- Income is the most important feature (~51%)  
- Location plays a major role  
- Boosting improved performance
